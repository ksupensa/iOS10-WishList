# iOS10-WishList
Application to save on your phone what you wish you owned.

### Concepts used:
* CoreData
  * NSFetchedResultController
  * NSFetchRequest
  * NSSortDescriptor
* UITableView
* UIPickerView
* UINavigationController
* UIImagePickerController(For Pictures on Device)
* Segue
* UISegment
